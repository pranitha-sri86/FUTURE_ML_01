# 📊 Sales & Demand Forecasting (Task 1 - Machine Learning Internship)

## 🚀 Project Overview
This project focuses on predicting future sales using historical business data. The goal is to help businesses make informed decisions using data-driven forecasting techniques.



# 🎯 Objective
To build a Machine Learning model that can analyze past sales trends and forecast future demand.


## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset
- Superstore Sales Dataset (from Kaggle)



## ⚙️ Steps Performed

### 1. Data Cleaning
- Removed missing values
- Converted date column to proper format

### 2. Feature Engineering
- Extracted Year, Month, Day from date

### 3. Data Visualization
- Plotted sales trends over time

### 4. Model Building
- Linear Regression (baseline model)
- Random Forest (advanced model)

### 5. Model Evaluation
- Compared models using Mean Absolute Error (MAE)

---

## 📈 Results
- Random Forest performed better than Linear Regression
- Captured non-linear patterns in sales data



## 💡 Business Insights
- Sales vary across months indicating seasonal trends
- Forecasting helps businesses in inventory planning
- Helps reduce overstocking and stock shortages
